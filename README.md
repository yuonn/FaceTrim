# FaceTrim

アニメ調のキャプチャー画像から顔だけをトリムした画像フォルダを作成します．



## 使い方

1. ```./images```にキャプチャー画像を入れる
2. ```FaceTrim.py```を実行
3. ```./Faces```に顔を切り抜いた画像が保存される





## 注意

* ```./images```に入れた画像はリネームされます．
